# Лабораторная работа №1: Представление чисел и арифметика в двоичных кодах

## Введение
Данная лабораторная работа посвящена изучению способов представления чисел в компьютере и реализации базовых арифметических операций над ними без использования встроенных библиотек. В проекте реализованы:

- перевод десятичных чисел в прямой, обратный и дополнительный коды (32 бита);
- сложение и вычитание в дополнительном коде;
- умножение и деление в прямом коде (с точностью до 5 знаков после запятой);
- операции (+, –, *, /) над числами с плавающей точкой по стандарту IEEE‑754 (binary32);
- сложение в различных двоично-десятичных кодах (8421 BCD, 2421 BCD, 5421 BCD, Excess‑3, Gray BCD).

Все преобразования выполняются вручную над массивами бит, без использования встроенных функций перевода (кроме ввода/вывода).

## Цель работы
Получить практические навыки низкоуровневого представления чисел и моделирования работы процессора при выполнении арифметических операций.

## Структура проекта
Проект состоит из следующих модулей:

| Файл                     | Описание                                                                          |
|--------------------------|-----------------------------------------------------------------------------------|
| `binary_utils.py`        | Базовые функции для работы с двоичными массивами (сложение, сдвиг, сравнение и т.д.) |
| `binary_codes.py`        | Класс `BinaryCodeConverter` для представления числа в прямом, обратном и дополнительном кодах |
| `additional_arithmetic.py` | Класс `AdditionalArithmetic` – сложение и вычитание в дополнительном коде          |
| `direct_arithmetic.py`   | Классы `DirectMultiplier` и `DirectDivider` – умножение и деление в прямом коде    |
| `ieee754.py`             | Класс `IEEE754Binary32` – преобразование и арифметика чисел с плавающей точкой     |
| `bcd.py`                 | Классы для различных BCD‑кодов и класс `BCDAdder` для сложения                    |
| `main.py`                | Интерактивное меню для демонстрации работы всех операций                          |
| `test_all.py`            | Модульные тесты, покрывающие все перечисленные классы                             |
